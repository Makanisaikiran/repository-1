Commands used to create repo and push code

  1. git int
  2. git add .
  3. git commit -m "...."
  4. git remote add origin
  5. git push orogin ..branch name..

Commands used to create Branch

  1. git branch ..Branch name.. (creates a branch with give branch name)
  2. git checkout ..branch name.. (switch to branch)

  3. git checkout -b ..branch name.. ( used to both creating and switching into the branch)
